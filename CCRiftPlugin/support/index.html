CCRift Preview Plugin

Premiere Pro and After Effects compatible

Windows 64-bit

  1. Download
  2. Put in C:\Program Files\Adobe\Common\Plug-ins\7.0\MediaCore

Mac OS X

  1. Download
  2. Unzip
  3. Put in /Library/Application Support/Adobe/Common/Plug-ins/7.0/MediaCore

Usage

  1. This is a Transmit plugin: it emulates a preview monitor device (a window) where you can orbit around the panoramic video
  2. Only equirectangular frames (2:1 aspect ratio) are supported
  3. Right click in the plugin window for options
  4. You can assign a shortcut to quickly enable / disable the Transmit plugin as described here
