OmniPreview

Panoramic video preview plugin

OmniPreview makes it easy to preview panoramic videos while editing in Adobe Premiere Pro and Adobe After Effects.

OmniPreview displays the equiectangular frames wrapped on a sphere.

You can drag with your mouse to explore the panorama, change field of view and frame size.

How to Install

Windows 64-bit

  1. Download
  2. Put in C:\Program Files\Adobe\Common\Plug-ins\7.0\MediaCore

Mac OS X

  1. Download and Unzip
  2. Put in /Library/Application Support/Adobe/Common/Plug-ins/7.0/MediaCore

Documentation

  1. This is a Transmit plugin: it emulates a preview monitor device (a window) where you can orbit around the panoramic video
  2. Only equirectangular frames (2:1 aspect ratio) are supported
  3. Click on the gear icon in the plugin window for additional options
  4. You can assign a shortcut to quickly enable / disable the Transmit plugin as described here
